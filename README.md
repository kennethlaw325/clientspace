# ClientSpace

Simple client portal for freelancers — share files, track progress, manage revisions, and get paid.

## Tech Stack

- **Frontend**: Next.js 15 + TypeScript + Tailwind CSS + ShadCN UI
- **Backend**: Supabase (Auth + PostgreSQL + Storage + Realtime)
- **Payments**: Stripe
- **Hosting**: Vercel
- **Email**: Resend

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
